== Description ==

This plugin loads the GoSquared Tracking Code for GoSquared applications

== Installation ==

1. Create a new directory called `gosquared` in `/wp-content/plugins/` and upload `gstc.php` and `gs.css` to that directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= I don't have an account code. How do I get LiveStats? =

Check out LiveStats at http://www.gosquared.com/livestats and sign up for an account

= I can't get this thing to work. Where can I contact you? =
